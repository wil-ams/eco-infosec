QuickScan Resultaten

Snelle compliance assessments per directie

Over QuickScans

QuickScans zijn lichtgewicht assessments die kwartaalmaand worden uitgevoerd per directie. Resultaten worden automatisch opgeslagen vanuit de QuickScan tool via de PurpleFish API.

Integratie voor Developers

IBAT en QuickScan tools kunnen resultaten automatisch opslaan via de PurpleFish API:

// Van andere tool (IBAT/QuickScan):
window.PurpleFishAPI.addQuickScan({
  id: 'QS-2026-001',
  directie: 'DIR-001',
  datum: '2026-01-30',
  score: 85,
  status: 'voltooid',
  resultaten: { governance: 90, security: 80 },
  bevindingen: [...]
});

// Voor SDAT resultaten:
window.PurpleFishAPI.addSDAT({
  id: 'SDAT-2026-001',
  applicatie: 'APP-001',
  datum: '2026-01-30',
  findings: [...],
  complianceScore: 87
});

Tip: Open de browser console (F12) om de volledige API documentatie te zien met console.log(window.PurpleFishAPI)
